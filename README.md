# 🧠 NeuroCare AI

An AI-powered mental wellness assistant built using **FastAPI** and **Streamlit**. The application provides AI-driven conversations, therapist recommendations, and emergency support through third-party API integrations.

## 🚀 Features

- 💬 AI Therapist Chat
- 📍 Therapist Finder
- 📱 WhatsApp Integration
- 🚨 Emergency Support
- ⚡ FastAPI Backend
- 🎨 Streamlit Frontend

## 🛠️ Tech Stack

- Python
- FastAPI
- Streamlit
- Groq API
- Twilio API
- Google Maps API
- Git & GitHub

## ▶️ Getting Started

```bash
git clone https://github.com/anjali-sharma07/NeuroCare-AI.git
cd NeuroCare-AI

python -m venv venv
venv\Scripts\activate

uv sync
```

Run the backend:

```bash
cd backend
uvicorn main:app --reload
```

Run the frontend:

```bash
streamlit run frontend.py
```
